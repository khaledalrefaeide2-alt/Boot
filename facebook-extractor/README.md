# FB Extractor — مستخرج منشورات فيسبوك

موقع ويب احترافي لاستخراج منشورات صفحات فيسبوك من خلال [Apify Facebook Posts Scraper](https://apify.com/apify/facebook-posts-scraper).

## المميزات

- 🔑 **الربط بمفتاح Apify API** — يُحفظ محلياً في متصفحك فقط ولا يُرسل إلا إلى Apify مباشرة.
- 🗓️ **فلاتر البحث** — تحديد الفترة الزمنية (من / إلى) وعدد المنشورات المراد استخراجه (حتى 1000).
- 📄 **دعم عدة صفحات** — أدخل أكثر من رابط صفحة (رابط في كل سطر).
- 📊 **عرض عصري للنتائج** — بطاقات منشورات مع الصور والتفاعلات، إحصائيات إجمالية (منشورات / إعجابات / تعليقات / مشاركات).
- 🔍 **بحث وفرز** — بحث فوري داخل النتائج، وفرز حسب الأحدث / الأقدم / الإعجابات / التعليقات / المشاركات.
- ⬇️ **تصدير** — تنزيل النتائج بصيغة JSON أو CSV (متوافق مع Excel والعربية).
- 🌙 **وضع ليلي / نهاري** مع حفظ التفضيل.
- 📱 تصميم متجاوب يعمل على الجوال والحاسوب.
- 📡 **صفحة الرصد التلقائي** (`monitor.html`) — أضف منصاتك الإعلامية مرة واحدة، وسترصد الصفحة آخر منشورين من كل منصة تلقائياً كل 30 دقيقة، وتعرضها من الأحدث إلى الأقدم مع تمييز الجديد، وحفظ السجل في المتصفح وتصديره JSON. ملاحظة: الرصد يعمل ما دامت صفحة الرصد مفتوحة في المتصفح (الموقع بلا خادم خلفي).

## طريقة الاستخدام

1. افتح `index.html` في المتصفح (أو انشره على GitHub Pages / أي استضافة ثابتة).
2. أدخل مفتاح Apify API الخاص بك — تحصل عليه من [Apify Console → Settings → Integrations](https://console.apify.com/settings/integrations).
3. أدخل روابط صفحات فيسبوك، وحدّد عدد المنشورات والفترة الزمنية.
4. اضغط **بدء الاستخراج** وانتظر اكتمال العملية، ثم تصفح النتائج أو صدّرها.

## ملاحظات

- الموقع ثابت بالكامل (HTML/CSS/JS) بدون خادم خلفي — جميع الطلبات تذهب من متصفحك إلى `api.apify.com` مباشرة.
- استخراج المنشورات يستهلك من رصيد حسابك في Apify حسب تسعيرة الـ Actor.
- 🛡️ **حماية من الاستهلاك الزائد:** يُمرَّر عدد المنشورات المطلوب كسقف فوترة إلزامي (`maxItems`) تفرضه منصة Apify نفسها على التشغيل، فلن تُحاسَب على أكثر من العدد الذي حددته (× عدد الصفحات المدخلة) حتى لو حاولت الأداة جمع المزيد.
