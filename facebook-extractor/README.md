# FB Extractor — مستخرج منشورات فيسبوك

موقع ويب احترافي لاستخراج منشورات صفحات فيسبوك من خلال [Apify Facebook Posts Scraper](https://apify.com/apify/facebook-posts-scraper).

## المميزات

- 🔑 **الربط بمفتاح Apify API** — يُحفظ محلياً في متصفحك فقط ولا يُرسل إلا إلى Apify مباشرة.
- 🗓️ **فلاتر البحث** — تحديد الفترة الزمنية (من / إلى) وعدد المنشورات المراد استخراجه (حتى 1000).
- 📄 **دعم عدة صفحات** — أدخل أكثر من رابط صفحة (رابط في كل سطر).
- 📊 **عرض عصري للنتائج** — بطاقات منشورات مع الصور والتفاعلات، إحصائيات إجمالية (منشورات / إعجابات / تعليقات / مشاركات).
- 🔍 **بحث وفرز** — بحث فوري داخل النتائج، وفرز حسب الأحدث / الأقدم / الإعجابات / التعليقات / المشاركات.
- ⬇️ **تصدير** — تنزيل النتائج بصيغة JSON أو CSV (متوافق مع Excel والعربية).
- 🌙 **وضع ليلي / نهاري** مع حفظ التفضيل.
- 📱 تصميم متجاوب يعمل على الجوال والحاسوب.
- 🗄️ **قاعدة بيانات محلية على جهازك** — خادم SQLite صغير بدون أي مكتبات (مجلد `local-server/`): شغّله بنقرة واحدة وسيحفظ الموقع كل المنشورات المستخرجة والمرصودة تلقائياً في ملف دائم على حاسوبك، بدون تكرار. التفاصيل في `local-server/README.md`.
- 📊 **لوحة الرصد التفاعلية** (`dashboard.html`) — الصفحة الرئيسية لرصد منصاتك الرسمية: تضيف صفحاتك ومفتاح Apify، فترصد اللوحة منشوراتها دورياً (بمدة قابلة للاختيار)، وتعرض النتائج في **رسوم بيانية تفاعلية** (نشاط النشر عبر الزمن، والتفاعل حسب المنصة) مع بطاقات إحصائية، وتصفية حية حسب المنصة والفترة الزمنية، وبحث وفرز داخل المنشورات. تشارك بياناتها مع صفحة عرض القائمة وقاعدة البيانات المحلية.
- 📡 **صفحة الرصد (عرض القائمة)** (`monitor.html`) — أضف منصاتك الإعلامية مرة واحدة، وسيستخرج الموقع **منشوراً واحداً من كل منصة تلقائياً كل 30 دقيقة**، ويعرضها من الأحدث إلى الأقدم مع تمييز الجديد، وحفظ السجل في المتصفح وتصديره JSON. ملاحظة: الرصد يعمل ما دامت صفحة الرصد مفتوحة في المتصفح (الموقع بلا خادم خلفي).
- 🧠 **محرّك التحليل والتصنيف** (`analyzer.js`) — لكل منشور مستخرج يُنفَّذ تلقائياً **تصنيف هرمي كامل** يعمل محلياً في المتصفح دون إرسال أي بيانات للخارج:
  1. تحليل النص (المحتوى، النبرة، النية، المعنى الضمني، السخرية).
  2. تحليل المصدر (نوع الحساب، مؤشر المصداقية، التفاعل).
  3. **المشاعر الأساسية** (إيجابي / سلبي / محايد) ← **المشاعر الدقيقة** (فخر وطني، أمل، تضامن، غضب وتحريض، كراهية، تخويف اقتصادي، استفسار…).
  4. **تصنيف النية** (دعم صريح/ضمني، نقد بنّاء/هادم، تحريض مباشر/ضمني، تضليل، سخرية سامة/بنّاءة، إعلام محايد).
  5. **تحديد المجال** (سياسي، اقتصادي، أمني، اجتماعي، إعلامي/أخلاقي، جرائم إلكترونية، حقوق إنسان، منصات التواصل).
  6. **مقياس شدة المخالفة 1–10** (ضئيلة → حرجة) وفق معايير التصنيف الإيجابي/السلبي (تهديد السلم الأهلي، الجرائم الإلكترونية، مدونة السلوك الإعلامي، سياسات ميتا/X، حقوق الإنسان، استهداف الدولة/الاقتصاد/الأمن/المجتمع).
  7. **رفع العلم** (Flagging) مع بيان السياسات/القوانين المخالَفة.
  8. **توصية إجرائية** واضحة: مراقبة (MONITOR) / تحذير وتعليق (FLAG) / إزالة وإنذار (REMOVE) / حظر وإحالة قانونية (ESCALATE).

  تُعرض النتائج على كل بطاقة كشارات مضغوطة + لوحة تحليل كاملة قابلة للطي، مع **ملخّص إحصائي** أعلى الصفحة (توزيع المشاعر، عدد المنشورات المرفوعة العلم، متوسط الشدة) وتصفية حية حسب التصنيف. وتُحفظ حقول التحليل أيضاً في قاعدة البيانات المحلية.

## طريقة الاستخدام

1. افتح `index.html` في المتصفح (أو انشره على GitHub Pages / أي استضافة ثابتة).
2. أدخل مفتاح Apify API الخاص بك — تحصل عليه من [Apify Console → Settings → Integrations](https://console.apify.com/settings/integrations).
3. أدخل روابط صفحات فيسبوك، وحدّد عدد المنشورات والفترة الزمنية.
4. اضغط **بدء الاستخراج** وانتظر اكتمال العملية، ثم تصفح النتائج أو صدّرها.

## ملاحظات

- الموقع ثابت بالكامل (HTML/CSS/JS) بدون خادم خلفي — جميع الطلبات تذهب من متصفحك إلى `api.apify.com` مباشرة.
- استخراج المنشورات يستهلك من رصيد حسابك في Apify حسب تسعيرة الـ Actor.
- 🛡️ **حماية من الاستهلاك الزائد:** يُمرَّر عدد المنشورات المطلوب كسقف فوترة إلزامي (`maxItems`) تفرضه منصة Apify نفسها على التشغيل، فلن تُحاسَب على أكثر من العدد الذي حددته (× عدد الصفحات المدخلة) حتى لو حاولت الأداة جمع المزيد.
